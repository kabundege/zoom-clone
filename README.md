# zoom-clone

### How it Works
  ### 1. Clone the project 
  ### 2. In your terminal run : 
  - ```npm install```  to nstall dependecies
  - ```npm install -g peer``` for info [Click Here](https://www.npmjs.com/package/peer)
  - ``` npm start``` to run the server
  ### 3. open your browser with express server
